# 📖 [CHESS_FOR_WORLD] <a name="about-project"></a>

 
**[CHESS_FOR_WORLD]** is an ongoing project focused on developing a multiplayer chess platform using React.js, TypeScript, Node.js, Express.js, Docker, Tailwindcss, and Socket.io technologies.
